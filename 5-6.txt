# Jenkins Installation (Windows)

## Prerequisites

* Install Java JDK 21 or above
* Verify Java:

```cmd
java -version
javac -version
```

---

## Step 1: Download Jenkins

Download Jenkins MSI from:

[https://www.jenkins.io/download/](https://www.jenkins.io/download/)

Select:

* Windows
* MSI Installer

---

## Step 2: Install Jenkins

1. Run `jenkins.msi`
2. Click **Next**
3. Select Installation Folder
4. Click **Next**
5. Select:

```text
Run Service as LocalSystem
```

6. Click **Next**

---

## Step 3: Configure Port

Enter:

```text
8081
```

Click **Test Port**

If successful:

```text
Next
```

---

## Step 4: Install Java JDK (If Required)

Download:

[https://www.oracle.com/java/technologies/downloads/](https://www.oracle.com/java/technologies/downloads/)

Install JDK 21

Default path:

```text
C:\Program Files\Java\jdk-21
```

---

## Step 5: Configure Java Path

When Jenkins asks for Java Home:

Select:

```text
C:\Program Files\Java\jdk-21
```

Click:

```text
OK
Next
```

---

## Step 6: Finish Jenkins Installation

Click:

```text
Install
```

After installation:

```text
Finish
```

# Configure SonarQube in Jenkins

### Add Credentials

```text
Manage Jenkins
→ Credentials
→ Global Credentials
→ Add Credentials
```

Choose:

```text
Kind: Secret Text
```

Paste SonarQube token.

ID:

```text
sonar
```

Save.

---

### Configure SonarQube Server

```text
Manage Jenkins
→ System
→ SonarQube Servers
```

Add:

```text
Name: sonar
Server URL: http://localhost:9000
Credentials: sonar
```

Save.

# Program 5: Jenkins CI/CD Pipeline with SonarQube

### Create Pipeline

```text
Dashboard
→ New Item
→ Pipeline
```

Name:

```text
SonarPipeline
```

### Pipeline Script

```groovy
pipeline {
    agent any

    stages {

        stage('Checkout') {
            steps {
                git 'https://github.com/USERNAME/REPOSITORY.git'
            }
        }

        stage('Build') {
            steps {
                bat 'mvn clean install'
            }
        }

        stage('SonarQube Analysis') {
            steps {
                bat 'mvn sonar:sonar'
            }
        }
    }
}
```

(Basic version suitable for lab exams.)

### Run

```text
Build Now
```

Check:

```text
Console Output
```

Expected:

```text
Build Successful
SonarQube Analysis Completed
```

# Program 6: Build Pipeline Using Maven

### Create Pipeline Job

```text
Dashboard
→ New Item
→ Pipeline
```

### Configure Maven

```text
Manage Jenkins
→ Tools
→ Maven Installations
```

Add Maven.

### Pipeline Code

```groovy
pipeline {
    agent any

    stages {

        stage('Compile') {
            steps {
                bat 'mvn compile'
            }
        }

        stage('Test') {
            steps {
                bat 'mvn test'
            }
        }

        stage('Package') {
            steps {
                bat 'mvn package'
            }
        }

    }
}
```

### Run Pipeline

```text
Build Now
```

Expected:

```text
Compile SUCCESS
Test SUCCESS
Package SUCCESS
BUILD SUCCESS
```
